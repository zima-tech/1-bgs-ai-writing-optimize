# 智能写稿优化和校对

## 原需求描述

办公智能，以智能助手等专业智能体，打破数据壁垒，智能创作写稿，对文字校对实现人工智能校对

---

## 页面结构

```
- 智能写作页面
  - 模板选择区
  - 编辑器区域
  - 工具栏

- 文字校对页面
  - 文档上传/输入区
  - 校对结果展示区
  - 错误详情列表
```

## 功能列表

### 1. 智能写作

| 功能 | 描述 |
|------|------|
| 模板套用 | 选择模板（工作报告/会议纪要/通知/函等），填入关键信息，自动生成文稿 |
| 提纲生成 | 输入主题关键词，自动生成三级文章大纲 |
| 全文生成 | 输入主题、类型、字数要求，一键生成完整文章 |
| 内容续写 | 选中已有段落，续写后续内容 |
| 风格优化 | 将文稿转为正式/简洁/生动风格 |
| 摘要提取 | 输入长文，自动提取核心要点 |

### 2. 文字校对

| 功能 | 描述 |
|------|------|
| 错别字校对 | 标红错别字，显示正确建议 |
| 标点校对 | 标红标点错误，显示正确用法 |
| 格式校对 | 检测字体/段落/标题层级问题 |
| 语法校对 | 标红病句，显示修改方案 |
| 敏感词检测 | 标红敏感词，显示风险等级 |
| 逻辑一致性 | 检测时间/数字/称谓矛盾 |

## 交互说明

- 模板选择：左侧栏展示模板列表，点击后加载模板到编辑器
- 文稿输入：富文本编辑器，支持粘贴/手动输入
- 校对触发：点击"开始校对"按钮，右侧显示错误列表
- 错误定位：点击错误项，光标跳转至文档对应位置
- 批量修改：点击"一键修复"自动应用所有建议
- 单条修改：点击具体建议项单独应用

## 展示元素

```
校对结果展示区：
├─ 错别字 [2处] ▶
├─ 标点错误 [1处] ▶
├─ 语法问题 [3处] ▶
├─ 敏感词 [0处]
└─ 格式问题 [5处] ▶

展开项显示：
[位置] [错误内容] → [修改建议] [修改理由]
```

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 文稿标题 |
| content | string | 文稿内容 |
| type | enum | 文稿类型 |
| template_id | string | 模板ID |
| corrections | array | 校对结果列表 |
